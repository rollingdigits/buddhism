불교(佛敎)[Buddhism]

불교 핵심 키워드 : 괴로움 - 누구에게나 괴로움은 있다. 괴로움이 없는 사람은 없다.

불교란?

  • 불(佛) : 부처님
  • 교(敎) : 가르침
부처님의 가르침

부처님이란?

1. 일반명사 `붓다` : `깨달은 사람` 의미

  • 고유명사 X → 절대적인 신 X
  • 불교 이전에 이미 존재하던 단어
  • 붓다 → 불타(佛陀) [중국] → 부텨 [조선] → 부처 + 님 [한국]

2. 역사적 인물 `붓다`

  • 깨닫기 이전 불리던 이름 : 고따마 싯닫따 (빠알리어), 가우타마 싯다르타 (산스크리트어)
  • 깨달은 고따마를 가리키는 호칭 : 사끼야 무니 [빠알리어] = 샤키야 무니 [산스크리트어] = 석가모니 [한자] : `석가족의 성자` 의미

깨닫기 이전

  • 꼴리야 왕국[구리 왕국] : 꼴리야족[구리족]이 건국, 수도는 데바다하[천비성]
  • 사끼야 공화국[석가 공화국] : 사끼야족[석가족]이 건국, 수도는 카필라바스투[가비라성], 코살라 왕국의 속국
  • 룸비니 동산 : 양국의 수도 데바다하와 카필라바스투 사이에 있던 동산
  • 고따마의 어머니이신 마야 부인은 구리국 천비성 출신, 석가국왕과 혼인한 이후엔 석가국 가비라성에 거주
  • 고따마는 룸비니 동산에서 탄생 : 어머니 마야부인이 출산을 위해 친정으로 가던중 이곳에서 고따마를 낳음
  • 고따마는 석가국 가비라성에서 태자로서 성장 → 현 네팔 땅

깨달음

  • 태자로서 부족한 것 없던 고따마가 모든 것을 버리고 출가한 이유 : 생사[생노병사]의 괴로움을 극복하고자 - 왕이 된다 한들 결국 죽어버리는 것 아닌가
  • 고따마가 깨달음을 얻은 곳 : 보드가야/붓다가야 → 현 인도 비하르주 가야시
  • 말라 공화국의 쿠시나가르에서 사망 → 현 인도 우타르프라데시주 마을
[참고] 인도 우타르프라데시주 아요디아시 : 옛 아요디아[아유타(阿踰陀)] - 코살라 왕국의 초기 수도 → 금관가야 허황후의 고향 → 경상남도 김해시와 자매결연 (2000년)

깨달음이란?

부처님의 마지막 유훈 : 형성된 것들[존재하는 모든 것]은 소멸하기 마련인 법이다. 방일하지 말고 (해야 할 바를) 성취하라.
  • 고따마는 무엇을 성취하였는가? → 진리를 깨달음
  • 진리란? → 연기, 사성제 (사성제는 진리이면서 `가르침`의 핵심)
  • 부처님의 깨달음 `연기` : 존재하는 모든 것은 실체가 없다. 다만, 조건지어진 것, 원인과 결과, 상호 의존
  • 부처님의 가르침 `사성제` : 네 가지 성스러운 진리 → 숲 속의 모든 생명들의 발자국이 코끼리 발자국에 들어가듯이 부처님의 모든 가르침은 사성제에 포함된다.
연기

이것이 있을 때 저것이 있다. 이것이 일어날 때 저것이 일어난다. 이것이 없을 때 저것도 없다. 이것이 사라질 때 저것도 사라진다.

무명을 조건으로 의도적 행위들이, 의도적 행위들을 조건으로 알음알이가, 알음알이를 조건으로 정신/물질이, 정신/물질을 조건으로 여섯 감각장소가, 여섯 감각장소를 조건으로 감각접촉이, 감각접촉을 조건으로 느낌이, 느낌을 조건으로 갈애가, 갈애를 조건으로 취착이, 취착을 조건으로 존재가, 존재를 조건으로 태어남이, 태어남을 조건으로 늙음/죽음과 근심/탄식/육체적 고통/정신적 고통/절망이 생긴다. 이와 같이 전체 괴로움의 무더기가 발생한다. (S12:1)

무명이 남김없이 빛바래어 소멸하기 때문에 의도적 행위[行]들이 소멸하고, 의도적 행위들이 소멸하기 때문에 알음알이가 소멸하고, 알음알이가 소멸하기 때문에 정신/물질이 소멸하고, 정신/물질이 소멸하기 때문에 여섯 감각장소가 소멸하고, 여섯 감각장소가 소멸하기 때문에 감각접촉이 소멸하고, 감각접촉이 소멸하기 때문에 느낌이 소멸하고, 느낌이 소멸하기 때문에 갈애가 소멸하고, 갈애가 소멸하기 때문에 취착이 소멸하고, 취착이 소멸하기 때문에 존재가 소멸하고, 존재가 소멸하기 때문에 태어남이 소멸하고, 태어남이 소멸하기 때문에 늙음/죽음과 근심/탄식/고통/정신적 고통/절망이 소멸한다. 이와 같이 전체 괴로움의 무더기가 소멸한다. (S12:1)

사성제 = 네 가지 성스러운 진리
  1. 고성제 = 괴로움의 성스러운 진리
  2. 집성제 = 괴로움의 일어남의 성스러운 진리
  3. 멸성제 = 괴로움의 소멸의 성스러운 진리
  4. 도성제 = 괴로움의 소멸에 이르는 길의 성스러운 진리
  • 괴로움은 꿰뚫어 알아야 한다.
  • 괴로움의 일어남은 버려야 한다.
  • 괴로움의 소멸은 실현된다.
  • 괴로움의 소멸에 이르는 길은 닦아야 한다.
사성제를 가르친 이유
  1. 내가 최상의 지혜로 안 것들 가운데 내가 가르치지 않은 것이 훨씨 더 많다. 내가 가르친 것은 아주 적다.
  2. 그러면 나는 왜 가르치지 않았는가? 이것은 이익을 주지 못하고, 이것은 청정범행의 시작에도 미치지 못하고, 염오로 인도하지 못하고, 탐욕의 빛바램으로 인도하지 못하고, 소멸로 인도하지 못하고, 고요함으로 인도하지 못하고, 최상의 지혜로 인도하지 못하고, 바른 깨달음으로 인도하지 못하고, 열반으로 인도하지 못한다. 그래서 나는 이것을 가르치지 않았다.
  3. 그러면 나는 무엇을 가르쳤는가? 나는 이것은 괴로움이라고 가르쳤다. 나는 이것은 괴로움의 일어남이라고 가르쳤다. 나는 이것은 괴로움의 소멸이라고 가르쳤다. 나는 이것은 괴로움의 소멸로 인도하는 도닦음이라고 가르쳤다.
  4. 그러면 왜 나는 이것을 가르쳤는가? 이것은 참으로 이익을 주고, 이것은 청정범행의 시작이고, 염오로 인도하고, 탐욕의 빛바램으로 인도하고, 소멸로 인도하고, 고요함으로 인도하고, 최상의 지혜로 인도하고, 바른 깨달음으로 인도하고, 열반으로 인도하기 때문이다. 그래서 나는 이것을 가르쳤다.
  1. 그러므로 그대들은 '이것이 괴로움이다.'라고 수행해야 한다. '이것이 괴로움의 일어남이다.'라고 수행해야 한다. '이것이 괴로움의 소멸이다'라고 수행해야 한다. '이것이 괴로움의 소멸로 인도하는 도닦음이다.'라고 수행해야 한다.


가르침이란?

불교의 목적

  • 이고득락(離苦得樂) : 괴로움을 여의고 행복을 얻는 것
  • 쉬운 말로 행복을 누리는 것
  • 세상에서 말하는 괴로움 : 괴로운 `느낌`을 경험하는 것
  • 세상에서 말하는 행복 : 행복한 `느낌`을 경험하는 것

두 가지 행복

  1. 감각적 욕망의 행복 → 소유의 행복, 저열한 행복 : 달콤함은 적고 재난은 많다
  2. (감각적 욕망에서) 벗어남의 행복 = 삼매/선정의 행복 [탐욕과 성냄이 사라진 상태] → 무소유의 행복, 수승한 행복 : 계발해야 할 행복, 고요한 행복 - 행복의 요소가 많고 괴로움의 요소가 적다. but 영원하지 않다. (삼매 역시 형성된 것, 조건에 의해 만들어진 행복, 무상하다.)
  • 부처님이 말씀하신 행복은 `괴로움의 완전한 소멸`
  • 행복한 느낌( 즉, 보통 사람들이 생각하는 행복)조차도 괴로움이다.
  • 느껴진 것은 그 무엇이나 괴로움의 속성이 있다. - 느낌 자체가 괴로움이다.

세 가지 행복

  1. 금생의 행복 = 인간의 행복 [감각적 욕망의 행복]
  2. 내생의 행복 = 천상의 행복 [감각적 욕망의 행복] - 선처[좋은 곳]에 태어나는 것
  3. 궁극적 행복 = 열반의 실현 [벗어남의 행복] - 지복(至福) = 영원한 행복, 완전한 행복
  • 금생의 행복과 내생의 행복은 주로 제가자를 대상으로 한 가르침
  • 궁극적 행복은 주로 출가자를 대상으로 한 가르침
열반 [무위법]
  • 궁극의 행복 - 열반을 실현하는 것이야말로 으뜸가는 행복
  • 네 가지 성스러운 진리 가운데 괴로움의 소멸의 성스러운 진리
  • 오온의 소멸 (S12:68/6 꼬삼비 경) ... (나는 오온으로 이루어져 있다. 오온은 무상하다. 무상한 것은 괴로움이다. 괴로움의 소멸은 열반이다.)
  • 탐욕의 소멸, 성냄의 소멸, 어리석음의 소멸 (S38:1 열반경)
  • 갈애의 소멸 - 갈애가 남김없이 빛바래어 소멸함, 버림, 놓아버림, 벗어남, 집착없음 (S56:11 초전법륜경)
  • 모든 형성된 것들[行]이 가라앉음, 모든 재생의 근거를 놓아버림, 갈애의 멸진, 탐욕의 빛바램[이욕(離欲)], 소멸 (S22:90/5 찬나 경) (S48:50/6 아빠나 경)
  • 열반은 하나이지만 그 이름은 모든 형성된 것들[행]의 이름과 반대되는 측면에서 여러 가지이다. 즉 남김없이 빛바램, 남김없이 소멸함, 버림, 놓아버림, 벗어남, 집착없음, 탐욕의 소멸, 성냄의 소멸, 어리석음의 소멸, 갈애의 소멸, 취착업음, 생기지 않음, 표상 없음, 원함 없음, 업의 축적이 없음, 재생연결이 없음, 다시 태어나지 않음, 태어날 곳이 없음, 태어나지 않음, 늙지 않음, 병들지 않음, 죽지 않음, 슬픔없음, 비탄없음, 절망없음, 오염되지 않음 (DA.iii.801)
  • 32가지 무위 즉 열반의 동의어 나열 (S43 무위 상윳따)
  • 43가지 열반의 동의어 (아비다나 빠디삐까)
  • 출세간도를 체험하는 순간 체득되는, 조건 지어지지 않은 상태 (초기불교이해)

세 가지 행복의 조건

금생의 행복의 조건

  • 학문과 기술 : 생업을 영위하며 오는 나의 행복, 가족의 행복, 사회의 행복
  • 보시와 지계 : 베품(봉사하는 삶), 계를 지킴(도덕적인 삶)
보시(베품, 봉사)란?
  • 행복의 원인
  • 증득의 뿌리
  • 부의 기반
  • 위험에 처한 자에게 기댈 곳 - 금생과 내생에 보시와 같은 의지처가 없다.
계(오계)
  • 불살생 : 살아 있는 생명을 함부로 해치지 말 것
  • 불투도 : 주지 않는 남의 물건을 갖지 말 것
  • 불사음 : 삿된 음행을 하지 말 것
  • 불망어 : 거짓말이나 헛된 소리(욕설 등) 하지 말 것
  • 불음주 : 술을 마시지 말 것 (취할 정도로 마시면 계를 어길 확률이 올라감)

내생의 행복의 조건

  • 보시와 지계 : 베풀고 계를 지키면 하늘에 태어난다 [시계생천]
  • 삼귀의 : 삼보에 대한 믿음

행복을 실현하는 토대 : 불법승 삼보에 대한 믿음(삼귀의)과 계를 지님(오계) (S55)

삼보
  1. 불 - 부처님 : 괴로움과 괴로움의 소멸을 가르친 깨달은 분
  2. 법 - 부처님의 가르침 : 사성제로 대표되는 진리
  3. 승 - 부처님의 가르침을 실천하는 수행자 : 괴로움의 소멸에 이르는 길의 성스러운 진리[도성제, 사성제]를 닦는 승려 집단

궁극의 행복의 조건

교학과 수행, 즉 이론과 실천을 통한 실현

  • 불교 이론 : 온처계근제연 → 나, 세계, 진리에 대한 부처님의 설명
  • 불교 수행 : 37보리분법 (=37조도품) → 직접 실천해야 할 가르침
  • 온처계근제연 : 오온, 육처, 18계, 22근, 사성제, 12연기
  • 37보리분법 : 사념처, 사정근, 사여의족, 오근, 오력, 칠각지, 팔정도 등 총 37가지 깨달음의 편에 있는 법들
  • 열반을 실현하는 것이야 말로 으뜸가는 행복 (SN267)
  • 바로 내 안에서 완전한 평화를 분명하게 안다. (D1 범망경)

순차적인 가르침

  1. 시 : 보시[베품]의 가르침, 가장 쉬운 가르침
  2. 계 : 계를 지킴의 가르침
  3. 생천 : 천상에 태어남의 가르침
  4. 감각적 욕망들의 재난과 타락과 오염원 → 출리[벗어남]의 공덕 → 중도수행[팔정도]
  5. 사성제를 앎 → 법안이 열림
